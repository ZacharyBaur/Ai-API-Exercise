AI Tool Effectiveness

I used Copilot with two models (GPT-4o and GPT-4o‑mini) to scaffold this project, generate fetch functions, and produce UI code. The AI was helpful for producing runnable code quickly and suggesting error handling and UX details. GPT-4o produced more detailed explanations and a fuller scaffold on the first try, while GPT-4o‑mini returned shorter, faster snippets that required a couple of follow‑ups to be complete. Example prompts and full responses were saved in my lab notes for comparison.

Note: Yo Momma API docs used for endpoints and query parameters.
